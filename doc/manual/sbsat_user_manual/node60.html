File Header

The header conforms to the DIMACS standard which is as follows:


p dnf <#vars> <#terms>


The letter 'p' must appear at the start of the header. The second header token, which in this case is 'dnf', identifies the input format. The header contains two integer fields: <#vars> is the number of distinct variables contained in the file; <#terms> is the number of terms contained in the file. The following is an example of a valid header for a file containing 97 terms composed from 56 distinct variables.


p dnf 56 97
