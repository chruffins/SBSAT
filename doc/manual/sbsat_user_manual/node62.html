XOR format

This input format is intended for specialized applications involving constraints that are exclusive-ors of conjunctions. Variables are represented as positive integers preceeded by the character x as in x45. All top-level Boolean functions are xors of conjunctions. A solution to the problem defined in a file is an assignment satisfying all functions. There are no manipulators or directives in this format.
